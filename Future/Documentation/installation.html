Installation


To install Future into your website visit our Downloads and download the version you would like to install
Once you have downloaded it, put the files into your websites directory then put the code below into your head tag

<link rel="stylesheet" href="./FutureFramework/style.css"/>
<script src="./FutureFramework/index.js"></script>

Example:

<!DOCTYPE html>
<html>
<head>
	<title>Example</title>
	<link rel="stylesheet" href="./FutureFramework/style.css"/>
    <script src="./FutureFramework/index.js"></script>
</head>
<body>	
</body>
</html>
